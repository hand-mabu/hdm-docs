# HDM产品手册

本文为HDM(Hand Data Management)数据采集平台的产品手册，简要介绍HDM各个功能模块


## 概述

HDM是为了解决用户在数据管理上面手工数据分散、数据质量差、反复对数、数据处理量大等问题而出现的。

HDM适用于企业预算管理、合并管理、商务智能、数据综合管理以及其他辅助应用，帮助企业快速完成**数据采集处理**。

> 适用于绝大部分场景
